# Camper Cafe Menu

A styled cafe menu webpage built with HTML and CSS.

## Preview

A coffee shop menu card featuring:

- Coffee section (French Vanilla, Caramel Macchiato, Pumpkin Spice, Hazelnut, Mocha).
- Desserts section (Donut, Cherry Pie, Cheesecake, Cinnamon Roll).
- Prices aligned to the right of each item.
- Footer with website link and address.

## Files

- `index.html` — page structure.
- `styles.css` — styling.

## How to Use

1. Clone or download the project.
2. Open `index.html` in your browser.

## Customization

To add or edit menu items, update the `<article class="item">` blocks in `index.html`:

- Change the `<p class="flavor">` or `<p class="dessert">` text for the item name.
- Change the `<p class="price">` value for the price.
- Add a new `<section>` block to introduce a new menu category.

To change the look, edit `styles.css`:

- `background-image` on `body` for the page background.
- `background-color` on `.menu` for the card color.
- `font-family` on `h1, h2` for the heading font.

## Technologies

- CSS3
