# Blog Post Card

A simple blog post card component built with HTML and CSS.

## Preview

A centered card on a pink background featuring:

- A cover image.
- A blog post title.
- A short excerpt summarizing the post.
- A "Read More" button with a hover effect.

## Files

- `index.html` — page structure.
- `styles.css` — styling.

## How to Use

1. Clone or download the project.
2. Open `index.html` in your browser.

## Customization

To make it your own, update `index.html`:

- Replace the `src` in `.post-img` with your own image.
- Change the `.post-title` text for your blog title.
- Update the `.post-excerpt` text with your summary.

To change the look, edit `styles.css`:

- `background-color` on `body` for the page background.
- `width` on `.blog-post-card` for the card size.
- `background-color` on `.read-more` for the button color.

## Technologies

- HTML5
- CSS3
