# CSS Projects

A collection of small projects built while learning and practicing CSS.

Each project explores different CSS concepts — from basic styling and typography,
to layout techniques like flexbox, to component design like cards, menus, and lists.
The projects grow in complexity over time, reflecting progress and experimentation
with what CSS can do.

## What's Inside

A set of mini frontend projects, each self-contained with its own HTML and CSS files.
Topics covered across the projects include:

- Selectors, specificity, and the cascade
- Typography and color
- Box model, spacing, and borders
- Flexbox layout
- Pseudo-classes and pseudo-elements
- Hover effects and transitions
- Responsive design basics

## Technologies

- HTML5
- CSS3
- Google Fonts
